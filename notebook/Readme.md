# Notebooks

본 폴더에는 서울형 키즈카페 입지분석 프로젝트의 주요 분석 과정을 담은 Jupyter Notebook 파일을 포함했습니다.

분석은 수요 추정, 군집화, 입지 최적화, 후보지 평가 순서로 진행했습니다.

파일 구성
파일명	설명
01_kmeans.ipynb	아동 수요를 반영한 가중치 K-means 분석
02_lscp.ipynb	수요지와 후보지를 분리한 LSCP 입지 최적화 분석
03_location_index.ipynb	후보지별 입지계수 및 평가 지표 산출
04_final_candidate_analysis.ipynb	최종 후보지 분석 및 우선순위 도출

분석 흐름
1. 가중치 K-means

일반 K-means는 모든 지점을 동일한 중요도로 반영합니다.

하지만 키즈카페 입지분석에서는 아동 수요가 큰 지역이 더 중요하게 반영되어야 하므로, 아파트 단지별 추정 아동 수요를 가중치로 반영했습니다.

이를 통해 단순한 위치 중심이 아니라, 수요 크기를 고려한 군집 중심점을 도출했습니다.

2. LSCP 입지 최적화

LSCP 분석에서는 데이터를 수요지와 후보지로 구분했습니다.

수요지: 어린이집, 유치원, 초등학교, 아파트 단지 등 키즈카페 서비스가 필요한 위치
후보지: 가중치 K-means로 도출한 군집 중심점과 기존 키즈카페 시설

후보지 데이터에는 기존 시설 여부를 나타내는 existing 컬럼을 추가했습니다.

기존 서울형 키즈카페와 민간 키즈카페는 이미 서비스를 제공하고 있으므로, LSCP에서 반드시 포함되도록 조건을 설정했습니다.

이후 기존 시설이 커버하지 못하는 수요지를 중심으로 신규 후보지를 선택하여, 아동복지 사각지대를 줄이는 방식으로 분석했습니다.

3. 후보지 평가

LSCP를 통해 도출된 후보지에 대해 다음 지표를 활용하여 평가했습니다.

대중교통 접근성
주차 편의성
사고다발지역 수
유해시설 수
기존 시설과의 거리
실제 설치 가능성

각 지표는 표준화 후 가중합 방식으로 최종 점수를 산정했습니다.
