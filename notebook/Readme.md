# Notebooks

본 폴더에는 서울형 키즈카페 입지분석 프로젝트의 주요 분석 노트북을 정리했습니다.

분석은 아래 순서로 진행했습니다.

```text
01_kmeans.ipynb
→ 02_lscp.ipynb
→ 03_location_index.ipynb
→ 04_final_candidate_analysis.ipynb
```

## 파일 설명

| 파일명 | 설명 |
|---|---|
| `01_kmeans.ipynb` | 아동 수요를 반영한 가중치 K-means 분석을 수행한 노트북입니다. 수요가 높은 지역이 군집 중심 형성에 더 크게 반영되도록 하여 신규 후보지의 기반을 만들었습니다. |
| `02_lscp.ipynb` | 수요지, 후보지, 기존시설을 분리하여 LSCP 입지 최적화를 수행한 노트북입니다. 기존시설은 반드시 포함되도록 조건을 설정하고, 기존시설이 커버하지 못하는 아동복지 사각지대를 보완할 신규 후보지를 도출했습니다. |
| `03_location_index.ipynb` | 후보지별 입지계수와 평가 지표를 산출한 노트북입니다. 대중교통, 주차, 사고다발지역, 유해시설 등 정책 기준을 반영했습니다. |
| `04_final_candidate_analysis.ipynb` | 최종 후보지 6곳을 분석하고 우선순위를 정리한 노트북입니다. 신규 후보지 반영 전후의 아동복지 사각지대 변화를 비교했습니다. |

## 주요 결과

- 서울형 키즈카페 신규 입지 후보 6곳 제안
- 기존시설 반영 전후 아동복지 사각지대 비율 비교
- 아동복지 사각지대 비율 8.5%에서 1.7%로 감소 확인
