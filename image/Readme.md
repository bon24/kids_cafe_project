# Images

본 폴더에는 서울형 키즈카페 입지분석 프로젝트의 핵심 시각화 자료만 선별하여 업로드했습니다.

전체 분석 과정에서는 대중교통, 교통안전성, 주차장, 금연구역, 유해시설, 사고다발지역 등 다양한 입지 평가 지표를 확인했습니다.  
다만 모든 지표를 이미지로 제공하면 시각화 자료가 과도하게 많아지고 핵심 흐름이 흐려질 수 있어, README와 결과 해석에 필요한 대표 이미지 3개만 포함했습니다.

세부 지표별 데이터프레임과 계산 과정은 `notebooks/` 폴더의 분석 노트북에서 확인할 수 있습니다.

<br>

## 포함 이미지

| 파일명 | 설명 |
|---|---|
| `candidate_score_comparison.png` | 최종 후보지별 교통편의성, 교통위험성, 교통안전성, 주차편의성 지수의 누적 점수를 비교한 그래프 |
| `kmeans_cluster_map.png` | 가중치 K-means를 통해 도출한 아동 수요 중심 군집 시각화 |
| `lscp_result_map.png` | LSCP 분석 결과 지도. 검정색 점은 수요지, 빨간 마커는 기존시설, 파란 마커는 군집 기반 신규 후보지를 의미 |

<br>

## 이미지 선별 기준

본 프로젝트에서는 다음 기준에 따라 대표 이미지만 선별했습니다.

1. 전체 분석 흐름을 설명하는 데 직접적으로 필요한가?
2. K-means → LSCP → 최종 후보지 평가 과정을 보여줄 수 있는가?
3. README에서 결과 해석을 보완하는 자료로 활용할 수 있는가?

금연구역, 교통안전성, 대중교통, 주차장 등 세부 평가 지표는 최종 점수 산정에 반영했지만, 개별 이미지로 모두 업로드하지는 않았습니다.  
해당 지표의 전처리 및 계산 과정은 노트북과 데이터프레임으로 확인할 수 있습니다.
