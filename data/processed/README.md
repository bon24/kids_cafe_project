# Processed Data

본 폴더에는 전체 분석 과정에서 사용한 전처리 데이터 중, 프로젝트 구조를 이해하는 데 필요한 대표 데이터 2개만 업로드했습니다.

전체 프로젝트에서는 어린이집, 유치원, 초등학교, 아파트 단지, 대중교통, 주차장, 기존 키즈카페, 유해시설, 사고다발지역 등 20개 이상의 데이터를 수집·전처리하여 사용했습니다.

다만 GitHub 포트폴리오에서는 가독성과 용량 관리를 위해 모든 원본 및 중간 데이터를 업로드하지 않고, 분석 흐름을 이해할 수 있는 핵심 전처리 데이터만 포함했습니다.

## 포함 파일

| 파일명 | 설명 |
|---|---|
| 아동교육시설 데이터.xlsx | 어린이집, 유치원, 초등학교 등 아동 수요와 관련된 교육시설 데이터를 정리한 파일 |
| 키즈카페 데이터.xlsx | 기존 서울형 키즈카페 및 민간 키즈카페 등 공급시설 데이터를 정리한 파일 |

## 데이터 공개 범위

본 저장소에는 전체 원본 데이터와 모든 중간 전처리 파일을 포함하지 않았습니다.

대신 수요지와 기존 공급시설을 이해할 수 있는 대표 데이터 2개를 제공하며, 전체 분석 과정과 데이터 처리 흐름은 `notebooks/kids_cafe_analysis.ipynb`에서 확인할 수 있습니다.
